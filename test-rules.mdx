## Common rules

This section includes examples that exercise general writing style rules shared across Vale styles.

- Avoid jargon such as `circle back`.
- Do not write `click here` in link text.
- Use a keyboard shortcut example like `Press Ctrl+C`.
- Use an Oxford comma example: `apples, oranges and bananas`.
- Avoid passive voice: `The feature was implemented by the team.`
- Avoid second person: `You should update the settings.`
- We need to circle back on this topic.
